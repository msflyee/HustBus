HUSTBus
========
# Production
HUSTBus是一个微信小程序，主要为了解决目前学校校车服务在不同时段冗余和紧缺的问题。
## Functions
* 在地图上显示正在运行的校车的实时位置
* 根据用户选择将出发站点的等车人数呈现给用户
* 失物招领
* 更多功能有待思考...
